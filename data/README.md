# Dataset Description

Dataset sử dụng trong dự án là **NF-UQ-NIDS-v2**.

- `raw/`: Dữ liệu gốc, chưa qua xử lý
- `processed/`: Dữ liệu đã được làm sạch, chuẩn hóa và sẵn sàng cho mô hình

Lưu ý:
- Không chỉnh sửa trực tiếp dữ liệu trong thư mục `raw/`
- Các file trong `processed/` được tạo ra từ notebook tiền xử lý
